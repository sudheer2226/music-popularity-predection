# 🎵 Music Popularity Prediction

> Machine Learning web application that predicts whether a song will become popular based on audio features like danceability, energy, tempo, and loudness.

## 🎯 Problem Statement

Music popularity is influenced by multiple audio characteristics.
This project analyzes song features to predict popularity using Machine Learning models and real-time user input.

## ⚡ Quick Demo

Input: Danceability=0.82, Energy=0.76, Tempo=120, Loudness=-5.4
Output: 🎶 Popular Song Prediction (High Popularity Probability)
